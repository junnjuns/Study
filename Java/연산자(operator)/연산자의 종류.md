## 비트 연산자 &, |, ^, ~, << , >> 

- &:  피연산자 둘다 1이면 결과는 1 , 그 외 0
- |: 피연산자 중 하나라도 1이면 결과는 1, 그 외 0
- ^: 피연산자 값이 서로 다르면 결과는 1, 그 외 0
- ~: 피연산자를 2진수로 표현했을 때 값을 반대로 바꾼다.
- <<:  2진수로 표현했을 때 n 만큼 왼쪽으로 이동 -> 원래값에서 2의 n 제곱을 곱한다.
- \>>: 2진수로 표현했을 때 n 만큼 오른쪽으로 이동 -> 원래값에서 2의 n 제곱을 나눈다.
